# Operational Deployments
Perfomrs daily automated tasks operations

## Overview
This repository demonstrates how PowerShell scripts can be orchestrated through automation to perform daily ICT operational checks.


## What Is Automated
- Daily execution of health checks
- Automated report generation
- Centralised logging
- Exception-only human review

---

## Automation Trigger
The automation is designed to run daily via:
- Scheduled task
- Power Automate
- RPA trigger
